# Changelog

All notable changes to **Nexus UI Kit** are documented here.
The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

---

## [1.0.0] — 2026-07-05
### Initial release 🎉
**Added**
- 28-page **Admin Dashboard** suite — analytics, tables, forms, auth, settings, profile, billing and more
- 6-page **SaaS Landing** — home, pricing, blog, about, contact, 404
- **Components Gallery** — 40+ interactive components in one reference page
- **Theme Showcase** — the demo hub with live, embedded previews
- **Documentation** — full guide with scroll-spy sidebar
- **Mobile Views** — key screens framed in live phone mockups
- Instant **light / dark** mode across every screen
- Three accent skins: **Violet, Emerald, Orange**
- ApexCharts integration — 6 chart types themed to the accent
- Zero build step — plain HTML, CSS and vanilla JS
